5. FAQ

Q: What should I do if I encounter a 500 Internal Server Error?

A: Check your server logs for more details. Common causes include incorrect file permissions, misconfigured .env file, or missing dependencies.

Q: How do I update Vontres AI?

A: Pull the latest changes from the repository, run composer update and npm install, then php artisan migrate to apply any new database changes.

Q: Can I use Vontres AI with a different database system?

A: While MySQL is recommended, Vontres AI can work with other database systems supported by Laravel, such as PostgreSQL or SQLite. You'll need to update your database configuration accordingly.
